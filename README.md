# Conway's Game of Life

## Dependencies:
- `tkinter`

## How to use:
Type in `python3 Conway.py` into the terminal.

## Interface:
- Play
- Pause
- Enable Customization (lets the user edit the current 'generation')
- Game Speed Slider (1/sec to 10/sec)
